# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our standards

We expect contributors and maintainers to be respectful, assume good intent, and use inclusive language. Personal attacks, harassment, and discriminatory behavior are not acceptable.

## Enforcement

Report unacceptable behavior to the maintainers:

- **GitHub Issues:** open an issue with the `code-of-conduct` label and ask for a private follow-up if needed (do not include sensitive personal details in the public thread).
- **Security or safety concerns:** use [GitHub Security Advisories](https://github.com/isonlycoolie/ArcFlow/security/advisories) for matters that must not be public.

Maintainers will review reports promptly and may warn, restrict, or remove contributors who violate this code.
