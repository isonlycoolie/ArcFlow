# Code of Conduct

This project follows the Contributor Covenant Code of Conduct. We expect all contributors and maintainers to uphold a welcoming, respectful, and inclusive community.

Summary:

- Be respectful and assume good intent.
- Use inclusive language and avoid personal attacks.
- Report unacceptable behavior to the maintainers at conduct@arcflow.dev (placeholder).

For the full text, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
