# ArcFlow trademark guidelines

The name **ArcFlow** and the ArcFlow logo identify the open-source project and its contributors. This repository is distributed under **MIT OR Apache-2.0**; those licenses grant rights to the **software**, not an unlimited right to use the project name in a way that implies endorsement.

## Permitted

- Refer to the project as "ArcFlow" when describing that you use, modify, or contribute to this software
- State that your product is "built with ArcFlow" or "based on ArcFlow" when accurate
- Use the logo in documentation that points to this repository or official project sites

## Requires permission

- Using "ArcFlow" in your product or company name in a way that suggests the project maintainers endorse you
- Registering "ArcFlow" or confusingly similar marks for competing products
- Modifying the logo or presenting a derivative mark as the official ArcFlow brand

For questions, open a GitHub issue with the `trademark` label or contact the repository maintainers listed on GitHub.
